# 🧠 Mental Health Prediction System

## 📌 Overview
This project is a Machine Learning-based web application that predicts a user's mental health risk level (Low, Medium, High) based on input features like stress level, sleep hours, and lifestyle factors. The system uses a trained Logistic Regression model and provides predictions through a Flask web interface.

---

## 🚀 Features
- Mental health risk prediction
- Simple and user-friendly interface
- Login and registration system
- Real-time prediction using trained model
